Setup & Installation
====================

Installation
------------

To use pyOPERA, first install it using pip:

.. code-block:: console

   (.venv) $ pip install pyopera

PyOPERA is based on a base Python package generated from a MATLAB project. It requires a MATLAB Runtime installation to work properly. MATLAB Runtime can be installed
in Windows, Linux, MacOS. It can also be installed from the command line using silent mode. For more information head over to the `MATLAB Runtime docs <https://www.mathworks.com/help/compiler/install-the-matlab-runtime.html>`_.
Download files can be found `here <https://www.mathworks.com/products/compiler/matlab-runtime.html>`_. This package was tested on MATLAB Runtime 9.12.
